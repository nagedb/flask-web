关于本项目

技术栈

  • Flask - 轻量级 WSGI Web 框架
  • Jinja2 - Python 模板引擎
  • Werkzeug - 全面的 WSGI Web 应用程序库

项目结构

flask-web/
├── app/
│   ├── __init__.py    # 应用工厂
│   ├── routes.py      # 路由定义
│   └── errors.py      # 错误处理
├── config.py          # 配置文件
├── run.py             # 启动脚本
├── requirements.txt   # 依赖列表
└── uploads/           # 上传文件目录

功能特性

  • ✅ 应用工厂模式
  • ✅ 蓝图模块化路由
  • ✅ Jinja2 模板继承
  • ✅ 错误页面处理
  • ✅ 文件上传功能
  • ✅ RESTful API
  • ✅ 多环境配置
